◆ checkCable_internal()

static bool checkCable_internal ( void  )
static

Method to check if the wire is broken.

This method enables the necessary GPIO clocks, sets the mode of the pins, checks the connection between them and also disables them at the end.

Returns
  • true - The connection is still okay.
  • false - The connection is broken!
