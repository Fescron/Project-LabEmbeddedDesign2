void ADXL_ackInterrupt ( void  )

Acknowledge the interrupt from the accelerometer.

Read a certain register (necessary if the accelerometer is not in linked-loop mode) and clear the static variable.

◆ ADXL_configActivity()

void ADXL_configActivity ( uint8_t  gThreshold)

Configure the accelerometer to work in activity threshold mode.

Route activity detector to INT1 pin using INTMAP1, isolate bits and write settings to both threshold registers.

Parameters
[in]gThresholdThreshold [g].

◆ ADXL_readValues()

void ADXL_readValues ( void  )

Read and display g values forever with a 100ms interval.

The accelerometer is put in measurement mode at 12.5Hz ODR, new values are displayed every 100ms, if an interrupt was generated a delay of one second is called.
