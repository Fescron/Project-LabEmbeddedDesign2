void error ( uint8_t  number)

Error method.

DEBUG mode
In debug mode this method flashes the LED, displays a UART message and holds the microcontroller forever in a loop until it gets reset. The error value gets stored in a global variable.

RELEASE mode
In release mode this method sends the error value to the cloud using LoRaWAN if the error number doesn't correspond to LoRaWAN-related functionality (numbers 30 - 50).

Parameters
[in]numberThe number to indicate where in the code the error was thrown.
